# 世界杯每日情报站

这是一个可部署的静态网页版本，用来展示每日世界杯赛果、赛前分析、预测比分和胜负角度。

## 本地预览

在当前目录启动一个静态服务，然后打开：

```sh
python3 -m http.server 8080
```

浏览器访问：

```txt
http://localhost:8080
```

## 部署为在线链接

可以把整个目录部署到任意静态网站平台，例如 Vercel、Netlify、GitHub Pages 或公司内部静态托管平台。

当前页面使用前端示例数据。若要接入真实每日数据，可以把每天 09:30 的自动汇总结果写入一个 JSON 文件或接口，再让 `app.js` 读取该数据。

### Vercel

把当前目录作为项目导入 Vercel，框架选择 `Other`，构建命令留空，输出目录填写 `.`。

### Netlify

把当前目录拖到 Netlify Deploys 页面，或连接 Git 仓库。构建命令留空，发布目录填写 `.`。

### GitHub Pages

把这些文件提交到 GitHub 仓库，在 Pages 设置里选择从仓库根目录发布。
